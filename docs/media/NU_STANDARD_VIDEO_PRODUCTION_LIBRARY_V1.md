# Nu Standard — Video Production Library (V1)

**Purpose:** Structured prompts and sequencing for hero loops, educational micro-videos, trust reels, operational B-roll, and reassurance clips—aligned with **calm premium** motion discipline.

**Master philosophy:** Motion serves **clarity** and **trust**, not spectacle. Prefer **stillness with intentional micro-movements** over kinetic saturation.

---

## Global motion governance

### Allowed camera motion

| Technique | When OK | Max amplitude |
|-----------|---------|---------------|
| Slow dolly-in | Hero reveal ≤ 8% depth change | Subtle |
| Locked tripod micro drift correction | Educational macro only | Imperceptible post-stabilize |
| Slider 24–36 inch lateral | Environmental establishing | One axis only per shot |
| Gentle pan ≤ 8° | Exterior establishing dawn | Single continuous ease-in-out |
| Handheld **with** heavy stabilization | Behind-scenes operational only | Must feel “quiet documentary” |

### Forbidden camera motion

- Whip pans, crash zooms, roll spins, drone orbit showreel clichés (unless **single** restrained ascending reveal explicitly briefed P0 hero exception).  
- Speed ramps except **uniform 100%** editorial policy (no ramp curves).  
- Hyperlapse urban glitter.

### Editing style allowed

- Straight cuts 90%; **single** cross-dissolve ≤ 12 frames for temporal continuity only.  
- L-cuts on ambient natural sound beds acceptable.  
- Color grade: restrained LUT aligned with still palette (warm neutrals + gentle teal shadow separation—not tealorange blockbuster).

### Editing / pacing forbidden

- Beat-sync flashy montages every 0.5s.  
- Whoosh SFX on every cut.  
- Flash-frame typography stomps.  
- Artificial zoom pulses on static UI screen recordings.

## Subtitles / captions rules

| Rule | Specification |
|------|-----------------|
| Timing | Max 2 lines; ≤ 42 chars/line domestic EN |
| Placement | Lower third safe zone 16:9 + vertical safe margin mobile burn |
| Motion | Fade 6f in/out only |
| Content | Plain language; **no** internal jargon (“dispatch”, “pipeline”, “ledger”) |
| Accuracy | Captions must match spoken word ≥ 98% WER target |

## Sequence V-PRC-001 — Process explanation calm (30–45s)

Structured beats:

1. **Intake honesty** (booking metaphor props only—never literal unreadable UI clone).  
2. **Estimate realism** (why structured factors—VO script separate legal review).

**Pacing:** Each beat ≥ 7s minimum dwell—**no** faster montage.
