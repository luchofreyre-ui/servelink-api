# Nu Standard — Image Prompt Library (V1)

**Purpose:** Production-grade, anti-drift prompts for bespoke imagery aligned with **premium trust**, **owner-operator accountability**, **calm professionalism**, **environmental clarity**, and **educational compression**.

**Global exclusions (append to every prompt negative block):** stock photo aesthetic; watermark; logos except Nu Standard approved marks when explicitly briefed; exaggerated HDR halos; neon teal cyber gradients; smiling forced corporate poses; cluttered MLS staging; visible generic brand cleaning bottles; text overlays unless art-directed; diversity stereotypes; unsafe ladder choreography; children identifiable without release path.

**Global positives:** photorealistic medium-format naturalism; neutral warm daylight (4500–5600K mixed); shallow depth where emotional; authentic residential architecture American suburban/transitional; subtle film grain optional ≤8%; color discipline — cream `#FFF9F3`, warm brass `#C9B27C`, restrained teal `#0D9488` accents **only as environmental bounce**, never neon UI blobs.

## Homepage hero imagery

### H-PROMPT-HER-001 — Exterior sunrise residence

**Placement:** `NSM-HER-001`  
**Environment:** Single-family transitional home, credible landscaping, dew-visible mild morning.  
**Lighting:** Soft directional sunrise camera-left; shadow roll gentle; no blown sky.  
**Framing:** 35mm equivalent landscape 16:9 safe focal center-weighted lower third door/walkway.  
**Emotional tone:** Arrival calm; premium restraint; not gated-mansion excess.  
**Realism:** architectural integrity; plausible gutters; realistic windows.  
**Wardrobe:** none prominent (optional distant figure ambiguous).  
**Home-style:** Midwest / coastal transitional hybrid acceptable; avoid cliché mansion columns.  
**Operational tone:** residential truth.

**Positive:** photoreal exterior, golden hour restraint, crisp foliage, honest siding texture.  
**Negative:** drone tilt cliché, saturated teal sky, McMansion caricature, snow unless seasonal brief.
