# Admin UI V1 Scope Cutoff

## In scope for v1

### 1. Dashboard
- recent activity
- urgent exceptions
- anomaly summary
- quick links into operational areas

### 2. Dispatch exceptions list
- filters
- sorting
- priority badges
- recommended action indicators
- click-through to booking detail

### 3. Booking dispatch detail
- booking summary
- dispatch timeline
- dispatch explainer
- operator notes
- manual assign
- manual redispatch
- exclude provider

### 4. Dispatch config control
- active config
- draft config
- edit draft
- compare
- publish preview
- publish
- publish history
- publish audit detail
- restore draft from audit

### 5. Activity feed
- list recent admin actions
- link into related context

### 6. Anomalies
- list
- basic filters
- audit visibility
- operational actions already supported by backend

---

## Explicitly out of scope for v1

- full BI/analytics dashboards
- advanced charting layer
- drag-and-drop dashboards
- dispatch simulation lab
- role management UI
- custom report builder
- export center
- user management console
- franchise owner admin tooling
- customer support suite
- bulk action framework across all admin areas
- visual workflow builder

---

## Scope guard rule

If a proposed feature does not directly improve:
- dispatch visibility
- dispatch intervention
- dispatch configuration safety
- admin auditability

it should likely not be in v1.
