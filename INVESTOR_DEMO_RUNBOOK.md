# Servelink Investor Demo Runbook

## Demo path
1. Show customer booking page
2. Show quote + secure checkout
3. Show booking timeline
4. Show admin booking operational detail
5. Show anomalies page
6. Show revenue readiness snapshot

## Key points to say
- Servelink is infrastructure, not a thin marketplace layer
- Payment state is tied to operational and financial control
- Trust, anomalies, and operational detail are part of the same system
- Nu Standard validates the platform in the real world

## Do not demo
- unfinished internal-only routes
- manual dev controls
- any route without live data or stable state
